# Healthcare Chatbot - Trained Dataset Integration

## Overview

The healthcare chatbot in `dashboard.html` has been enhanced to use **trained data from the healthcare dataset** (5,000 records containing symptom-disease-treatment relationships).

## Integration Summary

### 1. **Trained Knowledge Base**
- **Source**: `knowledge_base.json` (generated by `train_chatbot.py`)
- **Contents**: 15 unique symptoms mapped to 6 diseases with structured metadata
- **Location in Dashboard**: Hidden `<textarea id="knowledge-base-data">` containing trained KB as JSON

### 2. **Training Data Overview**

The trained dataset contains:
- **15 Symptoms**: pain, vomiting, headache, fever, nausea, chest, body, cold, cough, diarrhea, fatigue, runny, nose, sore, throat
- **6 Diseases**: Viral Infection, Migraine, Heart Problem, Food Poisoning, Common Cold, General Weakness
- **Advice Guidance**: Home care and doctor prescription recommendations for each symptom

### 3. **Symptoms ↔ Diseases Mapping**

```
Viral Infection:
  - fever, vomiting, nausea, runny, nose

Migraine:
  - headache, pain, body, cold, cough

Heart Problem:
  - chest, fatigue, diarrhea (high severity - EMERGENCY)

Food Poisoning:
  - sore, throat (medium severity)

Common Cold / General Weakness:
  - Similar to Viral Infection pattern
```

### 4. **Chatbot Workflow in Dashboard**

When a user types a health concern:

1. **User Input** → Types symptom name (e.g., "fever", "chest pain")
2. **Knowledge Base Search** → `searchSymptomGuidance()` function searches trained KB
3. **Match Found** → Returns structured guidance from trained data:
   - Detected disease/condition
   - Severity level
   - Home care advice
   - Doctor prescription guidance
   - Emergency flag
4. **Response Display** → Formatted message with:
   - Disease name
   - Recommended home care steps
   - Doctor prescription details
   - Emergency alert (if applicable)
5. **User Profile** → Also displays patient info (name, age, gender)
6. **Medication Extraction** → Parses and displays recommended medications

### 9. **How to Use the Trained Chatbot**

1. **Login** to dashboard at http://127.0.0.1:5000/dashboard
2. **Scroll to** "Health Assistant Chat" section
3. **Type a symptom** from the trained list:
   - Single symptoms: "fever", "chest", "headache"
   - Multi-word: "chest pain" (will match "chest")
   - Variations: "fever", "high fever" (matches "fever" keyword)
4. **View immediate response** from trained knowledge base
5. **See patient profile** automatically populated (if completed profile flow)
6. **Access prescriptions** extracted from trained guidance
7. **Emergency indicators** appear for high-severity conditions

### 11. **Advantages of This Integration**

✅ **Real-world Training Data**: Uses 5,000 healthcare records, not hardcoded rules
✅ **Fast Response**: No server round-trip for common symptoms
✅ **Structured Guidance**: Home care + Doctor prescription for each symptom
✅ **Emergency Awareness**: Automatically flags high-severity conditions
✅ **Patient Context**: Integrates with user profile (age, gender, surgeries)
✅ **Fallback Coverage**: If symptom not in trained KB, falls back to server chatbot
✅ **Confidence**: Each recommendation is backed by data distribution
